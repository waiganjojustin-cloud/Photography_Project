# Photography Project

## Student Information
**Name:** Justine Waiganjo

## Project Description
This project is a personal photography website created using HTML and CSS. It presents a short biography, an introduction to my understanding of the Matrix system, and a gallery of conceptual images with descriptive captions. The website demonstrates basic web design principles, including page structure, navigation, images, and styling.

## Features
- Personal biography
- Introduction section
- Conceptual image gallery
- Image descriptions
- Sidebar navigation
- Responsive HTML structure

## Technologies Used
- HTML5
- CSS3
